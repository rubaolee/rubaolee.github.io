From Bits to Brilliance

What is a Bit?

  • A bit is the smallest unit of data
  • It can be 0 (off) or 1 (on)

Binary Logic

  • Computers use binary operations
  • Common: AND, OR, NOT, XOR

What is XOR?

Exclusive OR: Returns 1 if inputs differ

ABA XOR B
000
011
101
110

XOR Properties

  • Commutative: A ^ B = B ^ A
  • Associative: A ^ (B ^ C) = (A ^ B) ^ C
  • Identity: A ^ 0 = A
  • Self-inverse: A ^ A = 0

Bitwise XOR in Code

a = 5  # 0101
b = 3  # 0011
a ^ b  # 0110 = 6

XOR Swap Trick

a = a ^ b
b = a ^ b
a = a ^ b

XOR in Encryption

cipher = plain ^ key
plain = cipher ^ key

Symmetric and fast

XOR in Checksums

Used in RAID, file integrity

Helps detect missing or changed data

Find Missing Number

res = 0
for i in range(1, n+1): res ^= i
for num in arr: res ^= num

Result is the missing number

Single Non-Duplicate

res = 0
for num in arr:
    res ^= num

Finds the number that appears only once
